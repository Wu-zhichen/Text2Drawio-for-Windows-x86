Text2Draw.io Windows x64 Portable 1.1.0

系统要求：Windows 10 或 Windows 11，64 位（x86-64）
分发形式：解压即用，无需另装 Python、Node.js 或 npm

主要内置组件：
- Text2Draw.io Manager（Electron 42.7.0）
- Text2Draw.io draw.io（基于 draw.io Desktop 30.3.14）
- CPython 3.12.10 Windows embeddable runtime
- Text2Draw.io Agent Server 与 figure skill

1.1.0 新增：draw.io 与插件全局中英文切换、插件内实时模型选择、
新版启动程序图标及双语 GitHub README。

本便携包使用的第三方组件仍分别受其原许可证约束。Electron、draw.io、
Python 及 Python 依赖的许可证文件或软件内许可页面随相应运行时提供。

当前可执行文件没有商业代码签名。首次启动可能出现 Windows SmartScreen
提示；仅在确认压缩包来源可信时选择“更多信息”→“仍要运行”。

API Key 与本地日志保存在：
%APPDATA%\Text2Drawio\
