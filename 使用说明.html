Text2Draw.io 一键使用说明

此版本已经内置 draw.io、Python 与全部运行依赖。不要移动压缩包内的单个子文件，也不需要安装 Node.js、npm 或 Python。

第一次使用

  1. 先将整个 ZIP 压缩包解压到普通文件夹，例如桌面。不要直接在压缩包预览窗口中运行。
  2. 双击顶层的 开始使用 Text2Drawio.cmd，也可以直接打开 Text2Drawio Manager\Text2Drawio Manager.exe。
  3. 在管理器中粘贴自己的 DeepSeek API Key，选择全局语言，然后点击“保存配置”和“测试连接”。
  4. 点击“一键安装”安装 draw.io 插件。
  5. 点击“启动并打开 draw.io”。以后再次使用时仍从管理器启动即可。
文件生成：打开 draw.io 左侧的 Text2Draw.io Agent 面板，可输入提示词，也可选择 PDF、Excel、Word、PowerPoint、CSV 或文本文件后生成可编辑矢量图。模型可直接在插件输入区下方实时切换。

语言设置

管理器中的全局语言会同时应用到 draw.io 和 Text2Draw.io 插件。保存后请完全关闭 draw.io，再由管理器重新打开。

可选的节点小配图

如需 AI 节点小配图，在管理器的“AI 配置”页填写 OpenAI API Key；不填写不会影响普通矢量图生成。密钥只保存在本机的 %APPDATA%\Text2Drawio\.env。

Windows 安全提示

当前分享版未购买代码签名证书，Windows SmartScreen 可能在首次打开时提示“Windows 已保护你的电脑”。确认压缩包来源可信后，点击“更多信息”→“仍要运行”。这不是运行故障。

故障排查

  1. 确保旧版 uvicorn / Text2Draw.io 服务没有占用 127.0.0.1:8765。
  2. 双击 验证Windows运行时.cmd 检查内置组件。
  3. 在管理器“安装与诊断”中打开日志；日志位于 %APPDATA%\Text2Drawio\service.log。
  4. 若更新 API Key，请保存后停止并重新启动本地服务，使新配置生效。

隐私说明

附件文本与提示词会发送到所选的 AI API 以生成图表；本地服务只监听 127.0.0.1。请勿上传无权处理的敏感文件。
